# LifeBot AI Chatbot

LifeBot is a small AI chatbot project created to practice Python, conversational logic, and basic AI chatbot development.

## 🚀 Features

- Simple chatbot interaction
- Python-based logic
- Notebook-based development
- Beginner-friendly AI project

## 🛠 Technologies Used

- Python
- Jupyter Notebook

## 📌 Project Purpose

This project was created as a learning project to understand how chatbot logic works and to improve Python programming skills.
